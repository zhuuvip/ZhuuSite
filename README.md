# ZhuuVIP — Vercel Deploy Guide

## Deploy in 3 Steps
1. Extract this zip → push to a new GitHub repo
2. Go to vercel.com/new → import the repo
3. Click Deploy — done!

## AI Chat
Works out of the box using a free AI backend (Pollinations.ai). No API key needed.

## Optional: Use Your Own OpenAI Key
Add in Vercel → Settings → Environment Variables:
OPENAI_API_KEY = sk-...
