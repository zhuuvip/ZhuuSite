# ZhuuVIP — Underwater Digital World

A premium multi-page web application with cinematic ocean theme.

## Pages
- **Home** — Premium SaaS landing page
- **Speed Test** — Real internet speed test with animated gauges
- **Portfolio** — Developer portfolio showcase
- **Community** — Free Fire gaming community with WhatsApp links

## Features
- Cinematic ocean canvas (bubbles, fish, light rays)
- Glassmorphism UI with neon cyan glow effects
- Background music player (Love Story - Indila)
- Fully responsive (mobile + desktop)

## Setup & Deploy to Vercel

### Local Development
```bash
npm install
npm run dev
```

### Deploy to Vercel
1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Framework: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Click **Deploy** — done!

No environment variables needed.
